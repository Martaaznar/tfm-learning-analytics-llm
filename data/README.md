# Data access note

The original data files used in this thesis are not redistributed in this repository.

The analysis uses the following source files from the dataset described in the thesis:

- `MainTable.csv`
- `CodeStates.csv`
- `Subject.csv`
- `early.csv`
- `late.csv`
- `DatasetMetadata.csv`

To reproduce the analysis, place the original dataset files in this `data/` folder using the same file names. The notebook assumes this folder structure unless paths are adapted manually.

The dataset is described in the thesis, and the repository focuses on documenting the analytical workflow, generated outputs and reproducibility materials.
